Moje Projekty

Ukázka mých NPM balíčků, Discord botů a Minecraft pluginů

InsaneKick

Minecraft plugin pro eventy a funkce při pádu.

Java Spigot
Tento plugin umožňuje spravovat eventy, zachytávat pád hráče a spouštět speciální efekty. Vhodné pro herní servery, které chtějí mít interaktivní a zábavné funkce.

FrontHelp

IntelliJ plugin pro pomoc při frontách a eventech.

Java IntelliJ
FrontHelp poskytuje nástroje pro efektivní správu front a eventů přímo v IntelliJ. Umožňuje rychle reagovat na situace a zlepšuje workflow.

Quanta

NPM balíček pro rychlé utility a backend funkce.

TypeScript Node.js
Quanta nabízí utility funkce pro Node.js, zjednodušuje práci s daty, soubory a API. Ideální pro rychlý start backend projektů.

ZikyZone

Discord bot s funkcemi pro moderaci a zábavu.

JavaScript Discord.js
ZikyZone je multifunkční Discord bot s moderačními příkazy, zábavnými hrami a statistikami pro komunitní servery.
